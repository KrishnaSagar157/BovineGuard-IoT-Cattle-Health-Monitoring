# BovineGuard – IoT Driven Cattle Wellness Tracker

BovineGuard is an IoT-based livestock monitoring system designed to track cattle health conditions and notify farmers in case of abnormal readings.

## Features
- Real-time cattle health monitoring
- Sensor-based data collection
- SMS alerts using GSM module
- Low-cost IoT solution for farmers

## Hardware Used
- ESP32-S3 microcontroller
- MAX30102 heart rate & SpO2 sensor
- MPU6050 accelerometer & gyroscope
- GSM module
- Power supply

## Software Used
- Arduino IDE
- Embedded C
- Serial communication

## System Workflow
1. Sensors collect physiological data from cattle
2. ESP32-S3 processes sensor readings
3. System detects abnormal conditions
4. GSM module sends SMS alert to farmer

## Applications
- Livestock health monitoring
- Smart farming
- Early disease detection
